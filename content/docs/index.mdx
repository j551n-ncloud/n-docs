## Learning Paths

<Accordions>

<Accordion id="beginner" title="üå± Beginner Path: First Homelab">

**Goal:** Set up a basic homelab with essential services

**Timeline:** 4-6 weeks

**Learning Path:**
1. **Week 1-2:** [Proxmox Installation](/docs/documentation/proxmox) - Set up your hypervisor
2. **Week 2-3:** [Basic VMs](/docs/documentation/vms-lxc) - Create your first virtual machines
3. **Week 3-4:** [Docker Basics](/docs/documentation/docker) - Learn containerization
4. **Week 4-6:** [Essential Services](/docs/software) - Deploy core applications

**Skills You'll Gain:**
- Virtualization fundamentals
- Basic networking concepts
- Container management
- Service deployment

</Accordion>

<Accordion id="intermediate" title="‚ö° Intermediate Path: Production-Ready Infrastructure">

**Goal:** Build scalable, monitored infrastructure

**Timeline:** 6-8 weeks

**Prerequisites:** Basic homelab experience

**Learning Path:**
1. **Week 1-2:** [Advanced Proxmox](/docs/documentation/proxmox) - Clustering and high availability
2. **Week 2-4:** [Kubernetes Setup](/docs/documentation/kubernetes) - Container orchestration
3. **Week 4-6:** [Monitoring & Logging](/docs/software) - Observability stack
4. **Week 6-8:** [Automation Scripts](/docs/scripts) - Infrastructure as code

**Skills You'll Gain:**
- High availability design
- Container orchestration
- Monitoring and alerting
- Infrastructure automation

</Accordion>

<Accordion id="advanced" title="üî• Advanced Path: Enterprise-Grade Setup">

**Goal:** Implement enterprise patterns and practices

**Timeline:** 8-12 weeks

**Prerequisites:** Solid infrastructure experience

**Learning Path:**
1. **Week 1-3:** Multi-site clustering and disaster recovery
2. **Week 3-6:** Advanced networking and security
3. **Week 6-9:** CI/CD pipelines and GitOps
4. **Week 9-12:** Performance optimization and scaling

**Skills You'll Gain:**
- Disaster recovery planning
- Advanced security practices
- DevOps methodologies
- Performance engineering

</Accordion>
